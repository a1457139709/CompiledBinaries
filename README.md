# .NET 4.8 Offensive Tool Builds

Precompiled .NET Framework 4.8 builds of various public offensive security tools that do not provide official releases.

## Purpose

This repository exists to provide ready-to-use binaries for tools that:

- Require manual compilation
- Lack official release packages
- Need legacy .NET Framework compatibility
- Are inconvenient to build repeatedly in lab environments

## Included Tools

- Rubeus
- SharpSuccessor
- (additional tools may be added over time)

## Build Environment

All binaries are compiled with:

- Microsoft .NET Framework 4.8
- Visual Studio 2022
- Release configuration
- x64 unless otherwise specified
